# RokuRC


Roku Remote Control

http://rokurc.com

Web Browser based Roku TV Remote Control with Power On/Off, Mute, Voume Up/Down Buttons

It does not use IR, so you can control the TV in the other room.

Works in Chrome, Safari, Edge, FireFox.
